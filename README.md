**署名：威软全网自动化工具**

## 功能特性

- 支持30+国内外主流网站自动签到
- 多平台部署：油猴脚本、浏览器插件、桌面软件、命令行工具
- 定时自动签到，确保连续性
- 签到记录保存，统计连续天数
- 支持多账号管理
- 失败自动重试机制

## 安装使用

### 油猴脚本

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Greasemonkey](https://www.greasespot.net/)
2. 点击安装脚本：`userscript/weiruan-checkin.user.js`
3. 访问支持的网站即可自动签到

### Chrome 扩展

1. 打开 Chrome 扩展管理页面 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `extension/chrome` 目录

### Firefox 扩展

1. 打开 Firefox 扩展调试页面 `about:debugging#/runtime/this-firefox`
2. 点击"临时载入附加组件"
3. 选择 `extension/firefox/manifest.json`

### 桌面软件

```bash
cd desktop
npm install
npm start
```

### 命令行工具

```bash
cd desktop
npm install

# 查看帮助
node src/cli.js help

# 配置站点凭证
node src/cli.js config jd

# 执行全部签到
node src/cli.js checkin

# 查看签到状态
node src/cli.js status
```

## 配置说明

### Cookie/Token获取方法

1. 在浏览器中登录对应网站
2. 按 F12 打开开发者工具
3. 切换到 Network（网络）标签
4. 刷新页面
5. 点击任意请求，在 Headers 中找到 Cookie

### 定时签到

- 油猴脚本：自动每日签到
- 浏览器扩展：可在设置中配置签到时间
- 桌面软件：支持开机自启和定时签到
- 命令行：可配合系统计划任务使用

## 注意事项

1. 请确保已登录对应网站
2. Cookie/Token 会定期过期，需要重新获取
3. 部分网站可能有频率限制，请勿频繁请求
4. 本工具仅供学习交流，请遵守各网站使用条款
